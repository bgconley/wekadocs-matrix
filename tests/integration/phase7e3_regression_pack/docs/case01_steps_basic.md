## Quick Recap

This guide walks you through installing **Widget Pro** quickly and safely.

## Step-by-step Installation

### Step 1: Prepare Environment

ORDER_STEP_ONE
- Ensure Python 3.11+ is installed.
- Set environment variables and create a virtual environment.
- Verify network access to package repositories.

### Step 2: Complete Installation

ORDER_STEP_TWO
- Run `widgetpro install`.
- Confirm services are running via `widgetpro status`.
- Create an admin user.

### Step 3: Validate Installation

- Execute `widgetpro diag` and review logs.
- Run a sample job to ensure end-to-end success.
