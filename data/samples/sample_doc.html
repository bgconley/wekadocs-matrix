WekaFS Quick Reference

This is a quick reference guide for WekaFS administrators.

Essential Commands

The most commonly used commands:

  • weka status - Check cluster status
  • weka mount - Mount a filesystem
  • weka cluster info - View cluster information

Configuration Files

Important configuration files to know:

File Purpose Location
weka.conf Main configuration /etc/weka/weka.conf
network.conf Network settings /etc/weka/network.conf
storage.conf Storage configuration /etc/weka/storage.conf

Common Procedures

How to Add a Node

Follow these steps to add a new node to your cluster:

  1. Install Weka on the new node
  2. Configure network connectivity
  3. Join the cluster: weka cluster join --ip=MASTER_IP
  4. Verify node status: weka cluster nodes

How to Create a Filesystem

Creating a new filesystem:

weka filesystem create my-fs \
    --capacity=10TB \
    --stripe-width=1M \
    --replica-count=2

Environment Variables

Set these environment variables for optimal operation:

  • WEKA_HOME=/opt/weka
  • WEKA_CONFIG_PATH=/etc/weka
  • WEKA_LOG_LEVEL=INFO
